\# Secure Video Communication System



\## Project Title

A Novel Hybrid Multikey Cryptography Technique for Secure Video Communication



\## Overview

This project implements a secure video communication system using hybrid cryptography techniques to protect video data during storage and transmission.



The system combines multiple cryptographic algorithms to provide confidentiality, integrity, and secure key exchange.



\## Technologies Used



\### Frontend

\- React.js

\- Vite

\- Tailwind CSS

\- Framer Motion



\### Backend

\- Java Spring Boot

\- Maven

\- MongoDB



\### Cryptography

\- AES-256-GCM for video encryption

\- ECC P-521 for secure key exchange

\- RSA-4096 for key wrapping

\- SHA-512 for hashing



\## Features

\- User authentication

\- Secure video upload

\- Chunk-based video processing

\- Hybrid encryption workflow

\- Secure video sharing

\- Encrypted video playback

\- Access controlled video communication
